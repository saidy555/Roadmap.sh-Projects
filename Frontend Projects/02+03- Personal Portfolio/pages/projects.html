HTML-Only Portfolio

I developed a personal portfolio website using HTML5 exclusively, without any CSS or JavaScript. The main goal was to demonstrate a strong understanding of core HTML structure, semantic elements, and accessibility best practices.

I prioritized using semantic HTML tags like <header>, <main>, <nav>, <section>, <footer>, <figure>, and <figcaption> to ensure the document's structure is meaningful and accessible to screen readers and search engines. All images were given descriptive alt attributes. This project highlights my ability to build a robust and well-structured web page from the ground up, a foundational skill for any web developer.

Interactive Calculator

This project is a fully functional calculator built using HTML5, CSS3, and JavaScript. The primary goal was to create an intuitive and responsive user interface for performing basic arithmetic operations. The frontend is entirely self-contained, requiring no external libraries or frameworks.

HTML Structure

The HTML file provides the foundational structure of the calculator. It's built with a main container (<div>) that holds the display screen (<input>) and the button grid. Each button is a <button> element with a specific class to identify its function (e.g., operator, number, clear). This semantic structure ensures that the layout is logical and easy to navigate.
